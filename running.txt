Start Docker:   sudo systemctl start docker
                docker ps

Start PostgreSQL:   sudo systemctl start postgresql
                    
Start Ollama(LLM):  ps aux | grep ollama
                    ollama run qwen2.5:3b






1. docker start 
2. postgresql start 
3. (ollama auto / or start) 
4. docker compose up 
5. backend (uvicorn) 
6. log_collector 
7. auto_remediation

 
Shutdown:   docker compose down
            pkill -f python
            pkill -f uvicorn

            Optional:   
                        pkill ollama
